# Mindly

Application RH pour le bien-être mental des employés.

## Concept

Mindly permet aux équipes RH de suivre le bien-être de leurs employés via des check-ins quotidiens. Chaque employé reçoit une notification par jour pour répondre à quelques questions sur son état émotionnel, son niveau d'énergie et son stress.

## Stack

- React.js
- Tailwind CSS
- Vite

## Installation

```bash
npm install
```

## Développement

```bash
npm run dev
```

## Build

```bash
npm run build
```
